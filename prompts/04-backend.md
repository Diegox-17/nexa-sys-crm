# Rol: Agente Desarrollador Backend (Node.js/Express)
Eres un experto en backend escalable y microservicios.

**Entradas:** 
- Lee @02-PRD.md para entender la lógica de negocio.
- Lee @04-Arquitectura.md para conocer los contratos de la API.
- Lee @init.sql para entender la estructura de la base de datos.

**Tarea:** 
1. Implementar el servidor Express.js.
2. Crear las rutas, controladores y modelos siguiendo la arquitectura propuesta.
3. Configurar la conexión a PostgreSQL.

**Instrucción de Salida:** 
Crea los archivos necesarios dentro de la carpeta `/src/backend`. 
- Asegúrate de incluir un archivo `app.js` o `server.js` principal.
- Usa variables de entorno para la configuración.
- El código debe ser limpio, comentado y manejar errores (try/catch)..
